# 千幕


🎬 一个 SillyTavern 第三方界面扩展，做你故事的**幕后导演**：埋线推演、递送创作灵感，不设定向日程与线性叙事路径，不替User做选择，最大化叙事自由度与演变弹性。

---

## ✨ 它能做什么

### 🎬 推演 —— 让世界自主运转

- 🪡 **暗线灵感池** · 推演下一幕的任务、因果链、角色动向与世界回声，提炼为线索后台注入正文，正文中自然触发取用。
- 🌍 **世界自运转** · NPC与事件各有独立运行逻辑，User可介入推动、可旁观见证、可全然不知；褪去主角光环，让故事拥有真实的生长质感。
- 🕯️ **伏笔显影**（可选）· 为每条暗线赋予完整跨幕生命周期（铺陈→升温→临界→高潮→落幕），推进节奏由正文呼应度自然驱动。
- 🏙️ **尘寰群生**（可选）· 搭建世界声景墙，容纳街角吆喝、坊间八卦、各阶级圈层琐碎，为世界铺就鲜活的环境底噪同时为User扩展视角落点。

### 🎭 幕外 —— 番外小剧场

- ✍️ **独立番外短剧** · 基于当前聊天相关设定生成支线、脑洞与小剧场，支持收藏归档；不同指令可存入「剧札」复用，兼具分类管理与导入导出功能。
- 🔒 **绝不污染正文** · 仅读取当前聊天作为背景参考，内容独立成篇，既不写入正文，也不纳入推演体系，正篇与番外互不干扰。
---

## 💡 好上手

- 🔌 **独立API**，与正文模型解耦，推演与小剧场模块均隔离运行，互不影响。
- 🧭 **零学习成本**，默认方案开箱即用，同时支持最大化自定义调整。
- 🎚️ **灵活可控**，注入项可逐项开关，不同小板块功能默认关闭，token占用与功能强度自由拿捏。
